#  Overview

CreditSetu AI is a modern AI-powered loan approval prediction platform designed to demonstrate real-world Machine Learning deployment using Streamlit.

The application predicts whether a loan application is likely to be approved, explains the prediction using Explainable AI (SHAP), and provides fairness analysis to help evaluate model bias.

The project follows a modular software architecture with complete separation of frontend and backend components, making it scalable and easy to maintain.

#  Installation

Clone the repository

```bash
git clone https://github.com/hardikkumar131206-source/CreditSetu.git
```

Move into the project

```bash
cd CreditSetu
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
